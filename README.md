# prueba_ReactJS
Prueba técnica de SPA con React.js

# Version nodeJs
v21.7.3

# Instalación
- clonar proyecto del repositorio.
- Validar que se tiene instalada y en uso la version correcta del nodejs que es v21.7.3 antes de usar el siguiente comando.
- usar comando "npm i" o "npm install"
- una vez instalada las dependencias puedes ejecutar el proyecto con el comando "npm run dev"
